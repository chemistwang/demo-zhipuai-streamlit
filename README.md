# 基于智谱 AI 和 streamlit 搭建的 FunctionCall 的 demo


- 模型底座：[智谱 AI](https://www.zhipuai.cn/)
- web 框架：[streamlit](https://streamlit.io/)

- 天气

- 发送短信
